# hackathon-2795-sun
Repo untuk proyek hackathon climate change

## Ide hackathon
Ide: website yang menampilkan cara menanam berbagai sayuran di pot di rumah. Tujuannya agar halaman rumah, walaupun sempit, menjadi hijau dan asri, serta memberi manfaat kepada penghuni rumah, yaitu mengurangi biaya belanja makanan karena di rumah sudah bisa panen tanaman sayuran. Ada gambar, ada video cara penanaman (ngambil dari Youtube), dan keterangan pupuk yang dipakai.
